### Exercício 1 
 Faça um algoritmo usando o for para mostrar os números pares e impares de 0 a 100.

 - #### Descrição:

Depois, foi utilizado o "if" com o "% 2 == 0", que é para ver se o resto da divisão dos números por 2 é igual a zero.
